# 🖥️ Inside a Computer System Lab (TryHackMe)

Today, I explored the basics of how a computer system works and how it boots up. At first, it seemed like simple theory, but I’m beginning to understand how important these concepts are for cybersecurity.

---

## 🧠 What I Learned

### 🔹 Core Components of a Computer System
I learned about the key parts that make up a computer:

- **CPU** – The brain that processes instructions  
- **RAM** – Temporary memory used while the system is running  
- **Storage (HDD/SSD)** – Where files and the OS are stored  
- **Motherboard** – Connects all components together  
- **PSU** – Supplies power to the system  

Understanding how these components work together gives me a clearer picture of how systems operate internally.

---

### ⚙️ The Boot Process
I also learned how a computer starts up:

1. Power is turned on  
2. BIOS/UEFI checks the system (POST)  
3. Bootloader is loaded  
4. Operating system starts  
5. System becomes ready to use  

---

## 🔐 Cybersecurity Insight

One key thing I learned is that the **boot process can be targeted by attackers**.

This means:
- Malware can run before the OS loads  
- Attackers can gain deep control of a system  
- These attacks can be hard to detect  

This made me realize that even the earliest stages of a system are important in security.

Even though this feels like basic knowledge now, it’s actually very important.
