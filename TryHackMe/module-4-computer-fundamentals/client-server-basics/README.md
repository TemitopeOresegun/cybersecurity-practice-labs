
# 🌐 Client-Server Basics – TryHackMe

## 📌 Overview
This lab covers how computers communicate over the internet using the **client-server model** and basic web concepts.

---

## 🧠 Key Concepts
- **Client** sends a request, **Server** responds  
- **HTTP(S)** is **stateless** (no memory of previous requests)  
- Websites use **cookies & sessions** to maintain login state  
- **HTTP Methods**: GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS, CONNECT, TRACE  
- **DNS** translates domain names to IP addresses  
- **Ports** identify services on a server  
- **Protocols** define communication rules  

---

## 🔐 Cybersecurity Insight
Understanding how systems communicate helps identify vulnerabilities like poor session handling and exposed services.
