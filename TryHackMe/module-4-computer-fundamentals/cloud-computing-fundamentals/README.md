
# ☁️ Cloud Computing Lab (TryHackMe)

## 📌 Overview
This lab introduces the fundamentals of cloud computing, including deployment models, service models, and hands-on experience with Amazon EC2.

---

## 🚀 Key Concepts

### 🔹 Deployment Models
- **Public Cloud** – Shared infrastructure over the internet  
- **Private Cloud** – Dedicated to a single organization  
- **Hybrid Cloud** – Combination of public and private  

### 🔹 Service Models
- **IaaS** – Rent infrastructure (e.g., servers, storage)  
- **PaaS** – Build apps without managing infrastructure  
- **SaaS** – Use software via the internet  

---

## 🛠️ Hands-on Experience
- Launched and explored a virtual server using **Amazon EC2**  
- Observed how cloud resources can be created and scaled on demand  

---

## 🌍 Why Cloud Computing?
- Scalable and flexible  
- Cost-efficient (pay-as-you-go)  
- High availability  
- Accessible globally
