
# ☁️ Cloud Computing Lab Notes (TryHackMe)

## 📌 Overview
This lab introduced the fundamentals of cloud computing, including deployment models, service models, and hands-on experience with launching a cloud server using Amazon EC2.

---

## 🚀 What is Cloud Computing?
Cloud computing is the delivery of computing services (servers, storage, networking, databases, etc.) over the internet, allowing users to access and scale resources on demand without owning physical infrastructure.

---

## 🔹 Cloud Deployment Models

### 1. Public Cloud
- Services are delivered over the public internet
- Shared infrastructure among multiple users (multi-tenant)
- Managed by third-party providers
- Example: AWS, Azure, Google Cloud

### 2. Private Cloud
- Dedicated infrastructure for a single organization
- Can be hosted on-premises or by a third party
- Offers greater control and security

### 3. Hybrid Cloud
- Combination of public and private clouds
- Allows data and applications to be shared between them
- Provides flexibility and optimized resource usage

---

## 🔹 Cloud Service Models

### 1. IaaS (Infrastructure as a Service)
- Provides virtualized computing resources over the internet
- Users manage OS, applications, and runtime
- Provider manages hardware and networking
- Example: Amazon EC2

### 2. PaaS (Platform as a Service)
- Provides a platform for developing, testing, and deploying applications
- Users focus on code without worrying about infrastructure
- Provider manages OS, runtime, and infrastructure

### 3. SaaS (Software as a Service)
- Software delivered via the internet
- No installation required
- Accessible via web browsers
- Examples: Gmail, Zoom

---

## 🛠️ Hands-on: Amazon EC2

### What is EC2?
Amazon Elastic Compute Cloud (EC2) is a service that allows users to create and manage virtual servers (instances) in the cloud.

### Key Actions Performed
- Launched a virtual server instance
- Explored instance configuration options
- Observed how resources can be scaled on demand
- Understood how cloud servers are provisioned quickly

---

## 🌍 Key Benefits of Cloud Computing

- **Scalability:** Easily increase or decrease resources
- **On-demand self-service:** Provision resources instantly
- **Pay-as-you-go pricing:** Only pay for what you use
- **High availability:** Reliable and fault-tolerant systems
- **Security:** Built-in tools and configurations
- **Global accessibility:** Access from anywhere in the world

---

## 💡 Key Takeaways

- Cloud computing removes the need for physical infrastructure management
- Deployment models define *where* services run
- Service models define *what level of control* users have
- EC2 demonstrates how quickly infrastructure can be deployed
- Flexibility and scalability are core advantages of cloud systems
