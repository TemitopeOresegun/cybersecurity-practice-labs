# 💻 Computer Types – Notes (TryHackMe)

## 📌 Overview
In this lab, I learned about different **types of computers** and how each is designed for a specific purpose.

Understanding this is important in cybersecurity because every device has **different strengths, limitations, and security risks**.

---

## 🧩 Types of Computers

### 💼 Laptops
- Screen and keyboard in one device  
- Portable for everyday computing  

🔹 **What makes it suitable:**
- Designed for **mobility and convenience**  
- Built-in battery allows use anywhere  

---

### 🖥️ Desktops
- Separate screen and keyboard  
- Used in a fixed location  

🔹 **What makes it suitable:**
- Provides **sustained performance**  
- Better cooling and upgrade options  

---

### 🧠 Workstations
- Screen and keyboard setup  
- Built for professional tasks  

🔹 **What makes it suitable:**
- High **precision and reliability**  
- Handles heavy workloads like:
  - Engineering  
  - Graphics  
  - Data processing  

---

### 🖧 Servers
- Provide services to multiple users over a network  

🔹 **What makes it suitable:**
- Designed for:
  - **Availability (runs 24/7)**  
  - **Handling many users at once**  
- Used for websites, databases, and file sharing  

---

### 🌐 IoT Devices (Internet of Things)
- Network-connected devices with a **single purpose**  

🔹 **Examples:**
- Thermostat  
- Smart doorbell  
- Fitness tracker  

🔹 **What makes it suitable:**
- Built for **connectivity and automation**  
- Efficient and lightweight  

---

### 🔧 Embedded Systems
- Computers built into another device  

🔹 **Examples:**
- Coffee maker controller  
- Automatic door sensor  
- Lamp dimmer chip  

🔹 **What makes it suitable:**
- Designed for **specific tasks only**  
- Fast and efficient with minimal resources  

---

### 📱 Smartphones
- Pocket-sized computers  

🔹 **Examples:**
- iPhone  
- Android phones  

🔹 **What makes it suitable:**
- Optimized for:
  - **Battery life**  
  - **Connectivity**  
- Combines communication and computing  

---

### 📲 Tablets
- Touch-first devices with larger screens  

🔹 **Examples:**
- iPad  
- Drawing tablets  

🔹 **What makes it suitable:**
- Designed for **visual interaction**  
- Good for media, creativity, and portability  

---

## 🔍 IoT vs Embedded Systems

| Feature        | IoT Devices 🌐                  | Embedded Systems 🔧           |
|----------------|-------------------------------|------------------------------|
| Connectivity   | Always network-connected      | Not always connected         |
| Purpose        | Data sharing & automation     | Device-specific control      |
| Flexibility    | Can be updated remotely       | Usually fixed function       |

🔹 **Key Insight:**
> All IoT devices are embedded systems, but not all embedded systems are IoT devices.

---

## 🔐 Cybersecurity Insight

### 💡 What I Learned:
- Each computer type has **different security risks**  
- Attackers target systems based on:
  - Weakness  
  - Accessibility  
  - Value  

---

### 🎯 Examples of Risks

| Device Type | Common Risks |
|------------|-------------|
| Laptops / PCs | Malware, phishing |
| Servers | Data breaches, service disruption |
| Smartphones | Malicious apps, unsafe networks |
| IoT Devices | Botnets, unauthorized access |

---

## 🧠 Key Takeaways

- Different computers are built for different purposes  
- Design choices affect **performance, security, and usage**  
- Understanding these helps in:
  - Identifying vulnerabilities  
  - Improving system security
