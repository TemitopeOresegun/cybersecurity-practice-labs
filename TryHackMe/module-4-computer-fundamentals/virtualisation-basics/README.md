
# 🖥️ Virtualisation Basics (TryHackMe Lab)

## 📌 Overview
This lab introduced me to **virtualisation**, which allows one physical computer to run multiple virtual systems. It helps improve efficiency, reduce cost, and makes scaling easier.

---

## 💡 Key Concepts

- **Virtualisation** → Running multiple systems on one machine  
- **Hypervisor** → Software that creates and manages virtual machines  
- **Virtual Machine (VM)** → A full virtual computer with its own OS  
- **Container** → A lightweight environment for running applications  

---

## ⚙️ VM vs Container

| Virtual Machine 🖥️ | Container 📦 |
|--------------------|-------------|
| Full OS included   | No full OS  |
| Slower             | Fast        |
| Heavy              | Lightweight |
| Strong isolation   | Less isolation |

---

## 🧠 What I Learned

- One physical server can run multiple systems using virtualisation  
- Hypervisors manage and isolate virtual machines  
- VMs provide flexibility and strong isolation  
- Containers are faster and more efficient for modern apps
