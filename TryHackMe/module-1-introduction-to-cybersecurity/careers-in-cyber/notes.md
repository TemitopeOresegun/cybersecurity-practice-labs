
# Careers in Cyber Notes

## Overview
Cybersecurity offers multiple career paths. Understanding roles helps plan learning and certifications.

## Key Roles
- **Security Analyst**: Monitors networks and systems for threats
- **SOC Analyst**: Works in SOC to respond to security incidents
- **Penetration Tester**: Tests systems for vulnerabilities
- **Security Engineer**: Builds and maintains secure systems
- **Red/Blue Team Member**: Offensive/defensive security exercises

## Skills & Certifications
- Networking fundamentals (CCNA, CompTIA Network+)
- Security fundamentals (CompTIA Security+, CySA+)
- Ethical hacking (CEH, OSCP)
- Tool familiarity: Nmap, Wireshark, Burp Suite

## Takeaways
- Cybersecurity is diverse; roles suit different interests
- Continuous learning is essential
- Labs and hands-on practice strengthen skills
