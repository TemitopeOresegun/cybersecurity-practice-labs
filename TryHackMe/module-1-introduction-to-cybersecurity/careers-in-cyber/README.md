
# Careers in Cyber - TryHackMe

## Overview
This lab provides an overview of the **cybersecurity career landscape**.  
It highlights roles, skills and career paths for anyone entering the field.

## Key Concepts Learned
- Common cybersecurity roles:
  - Security Analyst
  - Penetration Tester
  - Security Engineer
  - SOC Analyst
- Certifications and skills required
- Career progression in cybersecurity
- Importance of continuous learning
