# Defensive Security Intro Notes

## Overview
Defensive security is about **protecting systems, detecting threats and responding to attacks**. SOC teams are central to monitoring and defending an organization’s network.

## Key Concepts
- **SOC (Security Operations Center)**: Team that monitors and responds to security incidents
- **Threat Monitoring**: Watching logs, alerts and network activity for anomalies
- **Incident Response**: Steps to contain and remediate security incidents
- **Defense-in-Depth**: Layered security strategy

## Tools / Techniques Learned
- SIEM (Security Information and Event Management)
- IDS/IPS (Intrusion Detection/Prevention Systems)
- Log analysis and alerting

## Takeaways
- Defensive security is reactive and proactive
- Monitoring is critical for detecting attacks early
- Knowledge of attacks improves defense strategies
