## 📌 Core Terminology

### 🧭 Desktop
The main workspace after logging into Windows.

- Holds files, folders, and shortcuts  
- Displays wallpaper and icons  
- Used for quick access to important items  

---

### 📊 Taskbar
A control strip (usually at the bottom of the screen).

- Shows running and pinned applications  
- Contains the system tray (network, sound, time, notifications)  

---

### 🪟 Start Menu
The central hub for accessing your system.

- Open with the Windows key  
- Launch apps, access settings, and power options  

---

### 🔍 Search
Quickly find anything on your system.

- Search for apps, files, or settings using keywords  

---

### 📁 File Explorer
Tool for managing files and folders.

- Create, move, delete, and organize files  
- Navigate drives and directories  

---

### 🔄 Windows Update
Keeps your system up to date.

- Installs security patches and feature updates  

---

### 🛍️ Microsoft Store
Official Windows app marketplace.

- Download trusted apps  
- Apps update automatically  

---

### ⚙️ Windows Settings
Modern configuration panel.

- Manage system, devices, personalization, and privacy  

---

### 🧰 Control Panel
Legacy settings interface.

- Used for advanced configurations  

---

### 📈 Task Manager
Real-time system monitoring tool.

- View running processes  
- Monitor CPU, RAM, and disk usage  
- End unresponsive programs  

---

### 🛡️ Windows Security
Built-in security dashboard.

- Manages antivirus and system protection  

---

### 🔥 Windows Defender Firewall
Network protection feature.

- Blocks unauthorized access  
- Filters incoming and outgoing traffic  

---

## ✅ Key Takeaways
- Even familiar systems like Windows offer deeper insight when approached from a security perspective.
- Built-in tools play a critical role in monitoring and managing system behavior.
- Strong understanding of fundamentals is essential for cybersecurity work.
- System-level awareness is key for both defense and troubleshooting.
