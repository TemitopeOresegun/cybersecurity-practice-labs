
# 🌐 DNS in Detail – Notes

## 📌 Overview
The **Domain Name System (DNS)** translates domain names into IP addresses, enabling communication over the internet.

---

## 🏗️ Domain Hierarchy

### 1. Root Domain (.)
- Top-level of DNS hierarchy  

### 2. Top-Level Domain (TLD)
- gTLD: .com, .org, .net  
- ccTLD: .ng, .uk, .us  

### 3. Second-Level Domain (SLD)
- Main domain name (e.g., example in example.com)

### 4. Subdomain
- Example: www.example.com  

#### Rules:
- Max label length: 63 characters  
- Max domain length: 253 characters  
- Underscores not allowed in hostnames  

---

## 📄 DNS Record Types

### A Record
Maps domain to IPv4 address
