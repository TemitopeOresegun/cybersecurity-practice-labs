
# 🌐 DNS in Detail

## 📌 Overview
This project covers the **Domain Name System (DNS)** in detail, including domain structure, DNS resolution, and record types.

DNS translates human-readable domain names (e.g., example.com) into IP addresses used by machines.

---

## 🧠 Key Concepts Covered

- DNS and its role in networking  
- Domain hierarchy:
  - TLD (Top-Level Domain)
  - SLD (Second-Level Domain)
  - Subdomains  
- DNS Record Types:
  - A Record  
  - AAAA Record  
  - CNAME Record  
  - MX Record  
  - TXT Record  
- DNS Resolution Process  
- TTL (Time To Live)  
- Practical DNS configurations  

---

## 🧪 Practical Topics

- A Record configuration (IPv4 mapping)  
- CNAME configuration (domain aliasing)  
- MX Record setup (email routing)  
- TXT Record usage (verification & security)  

---

## 🚀 Why This Matters

Understanding DNS is essential for:
- Web hosting  
- Cybersecurity  
- Network troubleshooting  
- Email configuration
