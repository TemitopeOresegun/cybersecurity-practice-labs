
# OSI Model - TryHackMe

## Overview
Currently working on the **OSI Model** lab.  
Focuses on understanding the 7 layers of the OSI model and how data travels across networks.

## Key Concepts Being Learned
- The 7 layers of the OSI model:
  1. Physical
  2. Data Link
  3. Network
  4. Transport
  5. Session
  6. Presentation
  7. Application
- How data moves through each layer
- Importance of the OSI model for troubleshooting and cybersecurity
