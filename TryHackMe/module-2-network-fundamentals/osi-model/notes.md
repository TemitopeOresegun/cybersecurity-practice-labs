
# OSI Model Notes

## Overview
The OSI Model is a conceptual framework that describes how data moves across networks. Each layer has a specific role.

## Layers
1. **Physical** – Transmission of raw bits
2. **Data Link** – MAC addressing, error detection
3. **Network** – IP addressing, routing
4. **Transport** – TCP/UDP, reliable delivery
5. **Session** – Establishes and manages sessions
6. **Presentation** – Data formatting, encryption/decryption
7. **Application** – Interface for user applications

## Takeaways
- Understanding OSI layers helps troubleshoot network issues
- Layers assist in analyzing attacks and defending networks
- Labs will cover traffic examples for each layer
