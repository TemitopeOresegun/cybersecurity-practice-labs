## 🧠 Key Concepts Learned  

- **Port Forwarding**  
  - Mapping external traffic to internal services using routers  

- **Intranet**  
  - Private internal network used within organizations  

- **Firewalls**  
  - Packet inspection based on IP, port, and protocol  
  - Types:
    - Packet Filtering Firewall  
    - Stateful Inspection Firewall  
    - Application Layer Firewall  

- **VPN (Virtual Private Network)**  
  - Secure communication through encrypted tunnels  
  - Technologies:
    - PPP  
    - PPTP  
    - IPSec  

- **Networking Devices**  
  - Router (Layer 3 – Routing between networks)  
  - Switch (Layer 2/3 – Connecting devices)  
  - VLAN (Network segmentation)
