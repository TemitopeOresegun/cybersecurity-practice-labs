
# Intro to LAN Notes

## Overview
A Local Area Network (LAN) connects devices in a limited area like an office or home. Understanding LANs is critical for network security.

## Key Concepts
- **Network Devices**:
  - **Router**: Connects different networks
  - **Switch**: Forwards traffic within a LAN
  - **Hub**: Broadcasts data to all devices (legacy)
  - **Access Point**: Wireless connection to LAN
- **LAN Communication**:
  - Devices communicate using MAC and IP addresses
  - Frames and packets are sent to target devices
- **Network Topology**:
  - Star, Bus, Ring, Mesh
  - Star is common in modern LANs

## Takeaways
- LAN knowledge helps detect internal network attacks
- Device roles are critical to understanding traffic flow
- Lab practice reinforces network concepts
